## pycoterm - retro terminal for pyco

<img width="728" height="1082" alt="image" src="https://github.com/user-attachments/assets/fb009633-5f26-4e45-8579-d077d5837086" />

[Pyco](http://github.com/LeeHolmes/pyco) is a fully featured command-line calculator designed for everyday use and maximum utility. But if you want a fancy GUI app to use it with that can sit in your taskbar, a regular console window won't really cut it.

To solve this problem, pycoterm in the retro pyco terminal you're looking for. It automatically downloads pyco as a dependency and then makes it a treat to use.

### Features

Pycoterm supports:

- Cool retro CRT look
- Syntax highlighting of input
- Tab completion and traditional shell keyboard navigation
